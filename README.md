# SL-space | Elite Video Editors Marketplace

SL-space is a premium platform connecting content creators with elite freelance video editors. Elevate your production with top-tier talent in your city.

## 🚀 Live Demo
Once deployed, your link will appear here!

## ✨ Features
- **Dynamic Marketplace**: Browse and search for editors by skill, category, and location.
- **Premium Portfolios**: View high-quality work samples and detailed editor bios.
- **Real-time Authentication**: Secure login and registration for both Editors and Customers.
- **Admin Dashboard**: Full visibility into user registrations and platform activity.
- **Responsive Design**: Optimized for all devices with premium glassmorphism aesthetics.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Data Persistence**: JSON-based local storage (Production-ready for small scale)
- **Icons**: FontAwesome 6

## 📦 Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/salimtkms-cell/Salixframe-website.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the server**:
   ```bash
   node server.js
   ```

4. **Open in browser**:
   Visit `http://localhost:3000`

## 🛡️ Admin Access
To view user registrations and platform data, visit:
`http://localhost:3000/admin.html`
